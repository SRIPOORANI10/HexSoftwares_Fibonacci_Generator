# HexSoftwares_Fibonacci_Generator 

A simple yet powerful Python GUI application that generates Fibonacci series, analyzes statistics, and visualizes results using graphs.

---

## 🚀 Features

- Generate Fibonacci series up to N terms
- Display full series in GUI
- Show statistics:
  - Count
  - Sum
  - Maximum value
  - Even & Odd numbers
- Graph visualization using Matplotlib 📊
- Clean and simple Tkinter GUI
- Reset/Clear functionality

---

## 🛠️ Technologies Used

- Python 🐍
- Tkinter (GUI Development)
- Matplotlib (Graph plotting)
